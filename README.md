
# DND Campaign Manager

一个**极简、可用、可逐步扩展**的 DND 跑团资料管理工具。
本项目以“先能用，再慢慢改”为核心目标，不追求复杂架构，也不预设过多功能。

---

## ✨ 当前功能

* **多跑团管理**

  * 创建跑团
  * 删除跑团
  * 在不同跑团之间切换

* **固定分类管理**

  * 人物卡（characters）
  * 怪物卡（monsters）
  * 地图（maps）
  * 笔记（notes）

* **文件管理**

  * 导入任意文件（复制到对应目录）
  * 双击文件使用系统默认程序打开

* **地图功能（简化版）**

  * 存储地图图片
  * 简单缩略预览
  * 使用系统图片查看器查看原图

---

## 🧠 设计理念

* **不追求“大而全”**
* **不做复杂抽象**
* **不引入不必要的依赖**
* **优先保证功能真实可用**

这是一个**工程上“保守但可靠”的起点版本**，适合个人维护、长期迭代。

---

## 🛠 技术栈

* Python 3
* Tkinter（原生 GUI）
* Pillow（用于图片预览）
* 本地文件系统存储（无数据库）

---

## 📁 项目结构

```
dnd_manager/
├── main.py              # 主程序（当前所有逻辑集中于此）
└── data/
    └── campaigns/       # 跑团数据目录（首次运行自动创建）
        └── 跑团名/
            ├── characters/
            ├── monsters/
            ├── maps/
            └── notes/
```

* 每个跑团就是一个文件夹
* 不使用 ID、不使用数据库
* 文件结构即数据结构

---

## 🚀 运行方式

### 1. 安装依赖

```bash
pip install pillow
```

（Tkinter 为 Python 标准库，一般无需额外安装）

### 2. 运行程序

```bash
python main.py
```

首次运行会自动创建 `data/campaigns` 目录。
