# Implementation Plan: Character and Monster Templates

## Overview

为DND跑团管理器添加人物卡和怪物卡模板功能，同时进行基本的UI优化。采用最小侵入性修改，保持现有架构和功能完整性。

## Tasks

- [x] 1. 添加模板生成功能
  - 在App类中添加get_template_content方法
  - 根据分类返回对应的模板字符串
  - 人物卡模板包含：姓名、种族、职业、技能、装备、背景
  - 怪物卡模板包含：姓名、CR、属性、攻击、特性（默认为无）
  - _Requirements: 1.2, 2.2_
